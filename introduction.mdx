# Axion C SDK

The Axion C SDK is a lightweight, high-performance client library for the [Axion](https://axionquant.com) financial data API. It gives you direct access to market data, company financials, economic indicators, news, sentiment, SEC filings, and more — all from a simple C interface.

## How it works

The SDK wraps the Axion REST API using [libcurl](https://curl.se/libcurl/) for HTTP transport and [cJSON](https://github.com/DaveGamble/cJSON) for JSON parsing. Every API call returns an `AxionResponse` containing the parsed JSON, raw response data, HTTP status code, and any error message.

```c axion_example.c
#include "axion.h"
#include <stdio.h>

int main() {
    AxionClient *client = axion_init("your_api_key_here");

    AxionResponse *response = axion_stocks_prices(
        client, "AAPL", "2024-01-01", "2024-12-31", "1d"
    );

    if (response->error) {
        fprintf(stderr, "Error: %s\n", response->error);
    } else if (response->json) {
        char *out = cJSON_Print(response->json);
        printf("%s\n", out);
        free(out);
    }

    axion_response(response);
    axion_client(client);
    return 0;
}
```
