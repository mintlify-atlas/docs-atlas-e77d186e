## axion_indices_tickers

Returns a list of market index tickers, optionally filtered by exchange.

```c
AxionResponse *axion_indices_tickers(
    AxionClient *client,
    const char  *exchange
);
```

**Endpoint:** `GET /indices/tickers`

<ParamField path="client" type="AxionClient *" required>
  Initialized client returned by `axion_init`.
</ParamField>

<ParamField path="exchange" type="const char *">
  Exchange identifier to filter by (e.g. `"NYSE"`, `"LSE"`). Pass `NULL` to return indices from all exchanges.
</ParamField>

The `json` field of the response contains an array of index objects. Each object includes the ticker symbol, full index name, exchange, and region.

## axion_indices_ticker

Returns profile and metadata for a single market index.

```c
AxionResponse *axion_indices_ticker(
    AxionClient *client,
    const char  *ticker
);
```

**Endpoint:** `GET /indices/<ticker>`

<ParamField path="client" type="AxionClient *" required>
  Initialized client returned by `axion_init`.
</ParamField>

<ParamField path="ticker" type="const char *" required>
  Index ticker symbol (e.g. `"SPX"` for the S&P 500, `"NDX"` for Nasdaq-100). Case-insensitive on the server.
</ParamField>

The `json` field of the response contains an object with index metadata including the symbol, full name, number of constituents, exchange, and region.

## axion_indices_prices

Returns OHLCV price bars for a market index over a specified date range and time frame.

```c
AxionResponse *axion_indices_prices(
    AxionClient *client,
    const char  *ticker,
    const char  *from_date,
    const char  *to_date,
    const char  *frame
);
```

**Endpoint:** `GET /indices/<ticker>/prices`

<ParamField path="client" type="AxionClient *" required>
  Initialized client returned by `axion_init`.
</ParamField>

<ParamField path="ticker" type="const char *" required>
  Index ticker symbol (e.g. `"SPX"`).
</ParamField>

<ParamField path="from_date" type="const char *">
  Start date in `YYYY-MM-DD` format (e.g. `"2024-01-01"`). Pass `NULL` to use the server default.
</ParamField>

<ParamField path="to_date" type="const char *">
  End date in `YYYY-MM-DD` format (e.g. `"2024-12-31"`). Pass `NULL` to use the server default.
</ParamField>

<ParamField path="frame" type="const char *">
  Time frame for each bar. Accepted values: `"1d"` (daily), `"1h"` (hourly), `"5m"` (5-minute). Pass `NULL` to use the server default.
</ParamField>

The `json` field of the response contains an array of price bar objects. Each bar includes fields for the timestamp, open, high, low, close, and volume.

<Note>
  Market indices do not trade directly, but index price data reflects the calculated value of the index at each interval based on its constituent securities.
</Note>

## Example

```c
#include "axion.h"
#include <stdio.h>

int main(void) {
    AxionClient *client = axion_init("YOUR_API_KEY");
    if (!client) {
        fprintf(stderr, "Failed to initialize client\n");
        return 1;
    }

    /* Get S&P 500 index details */
    AxionResponse *ticker = axion_indices_ticker(client, "SPX");
    if (ticker->http_status == 200 && ticker->json) {
        printf("SPX: %s\n", ticker->data);
    } else {
        fprintf(stderr, "Error: %s\n", ticker->error ? ticker->error : "unknown");
    }
    axion_free_response(ticker);

    /* Fetch daily S&P 500 prices for 2024 */
    AxionResponse *prices = axion_indices_prices(
        client, "SPX", "2024-01-01", "2024-12-31", "1d"
    );
    if (prices->http_status == 200 && prices->json) {
        printf("SPX prices: %s\n", prices->data);
    } else {
        fprintf(stderr, "Error: %s\n", prices->error ? prices->error : "unknown");
    }
    axion_free_response(prices);

    axion_free_client(client);
    return 0;
}
```
