## axion_futures_tickers

Returns a list of futures contract tickers, optionally filtered by exchange.

```c
AxionResponse *axion_futures_tickers(
    AxionClient *client,
    const char  *exchange
);
```

**Endpoint:** `GET /futures/tickers`

<ParamField path="client" type="AxionClient *" required>
  Initialized client returned by `axion_init`.
</ParamField>

<ParamField path="exchange" type="const char *">
  Exchange identifier to filter by (e.g. `"CME"`, `"NYMEX"`). Pass `NULL` to return contracts from all exchanges.
</ParamField>

The `json` field of the response contains an array of contract objects. Each object includes the ticker symbol, underlying asset, expiration date, and exchange.

## axion_futures_ticker

Returns profile and metadata for a single futures contract.

```c
AxionResponse *axion_futures_ticker(
    AxionClient *client,
    const char  *ticker
);
```

**Endpoint:** `GET /futures/<ticker>`

<ParamField path="client" type="AxionClient *" required>
  Initialized client returned by `axion_init`.
</ParamField>

<ParamField path="ticker" type="const char *" required>
  Futures contract symbol (e.g. `"ESZ24"` for the December 2024 E-mini S&P 500). Case-insensitive on the server.
</ParamField>

The `json` field of the response contains an object with contract metadata including the symbol, underlying asset, expiration date, contract size, exchange, and open interest.

## axion_futures_prices

Returns OHLCV price bars for a futures contract over a specified date range and time frame.

```c
AxionResponse *axion_futures_prices(
    AxionClient *client,
    const char  *ticker,
    const char  *from_date,
    const char  *to_date,
    const char  *frame
);
```

**Endpoint:** `GET /futures/<ticker>/prices`

<ParamField path="client" type="AxionClient *" required>
  Initialized client returned by `axion_init`.
</ParamField>

<ParamField path="ticker" type="const char *" required>
  Futures contract symbol (e.g. `"ESZ24"`).
</ParamField>

<ParamField path="from_date" type="const char *">
  Start date in `YYYY-MM-DD` format (e.g. `"2024-09-01"`). Pass `NULL` to use the server default.
</ParamField>

<ParamField path="to_date" type="const char *">
  End date in `YYYY-MM-DD` format (e.g. `"2024-12-20"`). Pass `NULL` to use the server default.
</ParamField>

<ParamField path="frame" type="const char *">
  Time frame for each bar. Accepted values: `"1d"` (daily), `"1h"` (hourly), `"5m"` (5-minute). Pass `NULL` to use the server default.
</ParamField>

The `json` field of the response contains an array of price bar objects. Each bar includes fields for the timestamp, open, high, low, close, and volume.

<Warning>
  Futures contracts expire. Requesting price data beyond a contract's expiration date will return an error or an empty array. Use a continuous contract symbol if your data provider supports it.
</Warning>

## Example

```c
#include "axion.h"
#include <stdio.h>

int main(void) {
    AxionClient *client = axion_init("YOUR_API_KEY");
    if (!client) {
        fprintf(stderr, "Failed to initialize client\n");
        return 1;
    }

    /* List all CME futures contracts */
    AxionResponse *tickers = axion_futures_tickers(client, "CME");
    if (tickers->http_status == 200 && tickers->json) {
        printf("CME futures: %s\n", tickers->data);
    } else {
        fprintf(stderr, "Error: %s\n", tickers->error ? tickers->error : "unknown");
    }
    axion_free_response(tickers);

    /* Get details for December 2024 E-mini S&P 500 */
    AxionResponse *ticker = axion_futures_ticker(client, "ESZ24");
    if (ticker->http_status == 200 && ticker->json) {
        printf("ESZ24: %s\n", ticker->data);
    }
    axion_free_response(ticker);

    /* Fetch daily prices for ESZ24 over its final quarter */
    AxionResponse *prices = axion_futures_prices(
        client, "ESZ24", "2024-09-01", "2024-12-20", "1d"
    );
    if (prices->http_status == 200 && prices->json) {
        printf("ESZ24 prices: %s\n", prices->data);
    } else {
        fprintf(stderr, "Error: %s\n", prices->error ? prices->error : "unknown");
    }
    axion_free_response(prices);

    axion_free_client(client);
    return 0;
}
```
